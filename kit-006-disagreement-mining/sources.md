# The 12-Source List (verbatim from the issue's build)

## Cal Newport (4)
1. https://calnewport.com/getting-unremarkable-things-done-the-problem-with-david-allens-universalism/ — his direct critique of Allen; the single richest source in the run
2. https://calnewport.com/forget-chatbots-you-need-a-notebook/ — the analog-notebook / long-thinking position
3. https://calnewport.com/why-most-students-dont-understand-the-real-goal-of-note-taking/ — "Never Record Raw Information"
4. https://calnewport.com/in-defense-of-thinking-2/ — the digital-exile prescriptions

*Note: we first tried his New Yorker essay "The Rise and Fall of Getting Things Done" (2020) — it failed to process (paywall). Source #1 makes the same core argument on his own domain.*

8. https://fortelabs.com/blog/12-steps-to-build-a-second-brain/ — the mechanics, tools, and automation stack
