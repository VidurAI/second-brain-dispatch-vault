# The Vault — runnable kits from The Second Brain Dispatch

Every kit here is the actual tooling behind a published issue of [The Second Brain Dispatch](https://hemantpatil.substack.com) — a newsletter about real AI learning-system builds, written by someone who ships them. Not guides *about* tools: the prompts, agent briefs, and templates we actually ran, sanitized and packaged with honest cost/time numbers.

The essays are free and prove each kit works. The kits are free too — [subscribe](https://hemantpatil.substack.com/subscribe) if you want the next one in your inbox with the story of what it broke.

## How to use a kit

Each kit's README is a 15-minute setup runbook. Prerequisites are honest (most need any capable AI chat or agent runner; some steps are free-tier NotebookLM). Cost/time numbers in each README are from our own production runs, not estimates.

## The rules these kits are built on

1. **No fabricated numbers** — every statistic in the essays traces to a primary source, and kit 003 is literally the machine we use to enforce that on ourselves.
2. **Honest failure modes** — templates force AI to flag what it couldn't verify instead of sounding confident.
3. **Tested by us first** — each kit is our own production tooling; dogfooding is the QA.
